# fpga-alfredV2

## Project Overview: Accelerating Approximate Nearest Neighbour Search (ANNS)
Our project focuses on the hardware acceleration of Approximate Nearest Neighbour Search (ANNS) for high-dimensional vector retrieval, a bottleneck in modern Machine Learning workflows (such as vector databases, Retrieval-Augmented Generation, and similarity search).

We are investigating the acceleration of core steps in graph-based nearest neighbour retrieval. Specifically focusing on algorithms like HNSW (Hierarchical Navigable Small World). We are setting out to implement hardware-level optimizations in hardware for distance computation (e.g., Euclidean, cosine similarity, etc), candidate ranking, and top-k selection.
